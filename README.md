# 🚀 JCET Hub
### Smart Campus Management & Engagement Platform

## 📌 Overview
JCET Hub is a full-stack smart campus platform designed to streamline college operations, improve student engagement, and automate event and attendance workflows.

---

## ✨ Features

- 🎯 Event Management System (Create, Register, Track)
- 📲 QR-Based Attendance System
- 👨‍💼 Role-Based Dashboards (Admin, Faculty, Student)
- ⚡ Real-Time Updates using WebSockets
- 🏆 Campus Points & Activity Tracking
- 📄 Resume Builder for Students

---

## 🏗️ Tech Stack

- Backend: Node.js, Express.js
- Frontend: Tailwind CSS
- Database: PostgreSQL
- Real-Time: WebSockets

---

- REST APIs for communication
